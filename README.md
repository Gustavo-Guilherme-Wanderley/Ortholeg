# Ortholeg

<H3>O projeto Ortholeg tem a finalidade de criar um dispositivo de aúxilio a locomoção para pessoas que necessitam de ajuda quanto a isso. Sendo assim, nesse github contém todo o material necessário para replicar o hardware e software utilizado no exoesqueleto.</H3>
<img width="600" height="450" alt="Image" src="https://github.com/user-attachments/assets/084ba103-7497-4d97-9403-00b55f111cd1" />
<H3>A Ortholeg board V1 consiste em uma placa que contém dois arduinos nanos, um conversor buck para redução de tensão, dois conectores para conexão dos encoders e mais alguns outros para comunicação do driver dos motores.</H3>
